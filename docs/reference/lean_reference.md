A per-theorem status table for everything under
[`../../lean/ActinfPolicyEntanglement/`](../../lean/ActinfPolicyEntanglement/).
The status column is one of:

* **proved** — full proof, no `sorry`.
* **forwarder** — proved by forwarding to another theorem.
* **boundary** — type-checked statement whose closed-form / algebraic
  content is supplied as a hypothesis or threaded through definitional
  unfolding; the theorem certifies the boundary identity without any
  `sorry` of its own.
* **witness** — boundary form whose existence claim is supplied by a
  `structure` parameter; the theorem certifies the witness without
  any `sorry` of its own. Used for theorems whose analytic content
  (KL chain rule, real-analytic continuity, matrix SVD, tensor-train
  rank envelopes, measure tightness) is scoped to a separate Mathlib4
  analytic-discharge layer.
The retired `sketch` and `deferred` buckets appear only in revision
history. They are no longer legal current statuses in
`manuscript/refs/labels.yaml`; the status-table tests reject them.

The boundary fragment is **`sorry`-free, `axiom`-free, and
`unsafe`/`partial`/`noncomputable`-free**. Refresh hygiene with:

```bash
uv run python scripts/build_lean.py     # asserts the budget is 0
```

## What the Mathlib4 discharge layer adds

A separate `lean/MathlibProofs/` library now discharges the headline
real-valued decomposition through `free_energy_decomposition_full` and
the general-K finite-KL kernel `entanglement_decomposition_generalK`.
It remains separate from the stock-Lean boundary fragment, and
row-specific proof slices should import the boundary fragment and
discharge the remaining witness payloads there:

* `dualFlat_pythagorean_witness`'s caller can be derived from the KL
  chain rule over finite probability spaces.
* `BoundedQuadraticTax` instances can be constructed from Bregman
  Taylor expansion over real finite-dimensional charts.
* `SmallLambdaTolerance` instances can be constructed from
  real-analytic continuity at `λ = 0`.
* `FreeEnergyConvexityWitness` instances (Thm 5.6) can be supplied
  from log-partition convexity once the finite KL / Bregman bridge is
  in place.
* `LocalConcavityAtZero` instances (Prop 11.1) can be constructed
  from Taylor-expansion lemmas.
* `MarkovBlanketSeparationWitness` (Prop 19.3) is algebraic at the
  boundary; a Mathlib extension supplies the entropy / total-correlation
  numerical primitives.
* The Schmidt-rank-1 ⇔ mean-field characterization can be supplied by
  Mathlib4 matrix-rank and outer-product lemmas.

Critically, this composes **on top** of the boundary fragment without
touching the existing files: the witness boundary form is the API
contract.

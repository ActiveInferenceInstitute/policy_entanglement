# `docs/guides/styleguide/`

This directory contains the enforceable styleguide fragments behind
[`../styleguide.md`](../styleguide.md). The fragments are split by
artifact type so authors can check only the rules they are changing:

* [`prose.md`](prose.md) - American English, claim strength, and
  citation discipline.
* [`citations.md`](citations.md) - citation registry and bibliography
  rules.
* [`equations.md`](equations.md) - registered equation and reference
  conventions.
* [`figures.md`](figures.md) - figure registry, captions, metadata, and
  generation checks.
* [`hyperparameters.md`](hyperparameters.md) - numerical-value ownership.
* [`manuscript-variables.md`](manuscript-variables.md) - `[[VAR:...]]`
  token contract.

For live counts and current release status, use the generated reports
under `output/reports/`; this folder should describe rules, not maintain
volatile status numbers.
